Materia: Desarrollo de aplicaciones web.

Proyecto final
Implementar en PHP + MySQL + CSS + [demás herramientas] un sistema web que permita crear rutas turísticas.
Por ejemplo:
La ruta del mole – asociados restaurantes recomendados..
La ruta de la cerveza – asociar lugares para tomar cerveza..
Algunas ideas del sistema son:
Se pueden crear nuevas rutas combinando las existentes
Se pueden consultar las rutas que existen el sistema
Se debe tener usuarios para crear las rutas, habrá rutas similares pero de diferentes usuarios
Cada usuario solo puede modificar, dar de baja o dar de alta sus propias rutas
El administrador podrá consultar, modificar, eliminar los usuarios y las rutas
Un internauta puede consultar las rutas
Un internauta se puede volver un usuario del sistema
Las rutas deben estar acompañadas de imágenes, descripción, costos, fechas, recomendaciones, comentarios, etc.
